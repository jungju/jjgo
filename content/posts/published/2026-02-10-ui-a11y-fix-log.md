# UI 접근성 수정 로그

체크 도구 경고를 쌓아두면 나중에 고치기 더 어렵다.
작게 나눠서 바로바로 수정했다.

## 발견

- 클릭 이벤트가 있는 비상호작용 요소 존재
- 이미지 alt 누락
- 자기닫힘 태그 사용으로 마크업 경고 발생

## 수정

1. `div` 클릭 UI를 버튼/링크로 교체
2. 이미지에 의미 있는 alt 추가
3. `audio` 태그를 명시적 닫힘 태그로 변경

## 메모

- 기능보다 사용성 기본기부터 유지
- 경고 1개라도 PR 전에 정리
